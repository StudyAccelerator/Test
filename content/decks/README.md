# Sales-call decks

Slide decks Waleed keeps open live on sales and partnership calls. Built with the `deck-studio` skill (`.claude/skills/deck-studio/`): source HTML plus rendered 16:9 PDF live together in each folder, and the HTML is always the thing you edit. Rebuild a PDF with:

```
scripts/render-deck.sh content/decks/<slug>/deck.html "content/decks/<slug>/<Name>-Deck.pdf"
```

These are private sales materials. They are never published on the site, linked from it, or moved into `public/`. Waleed reviews every deck before it is used, and prices or dates shown here must match the live page they came from; when a page changes, the deck changes in the same session.
